核心 API
POST
/pub/task
建立 Task，支援立即執行（immediate）與延遲執行（delayed），回呼指定的 Callback API。
POST
/pub/event
發布 Event，系統根據訂閱者清單自動派發 Task，執行對應的 Callback。
POST
/register
訂閱 Event，將 Subscriber 端點與事件名稱綁定，後續 pub 時自動觸發。
